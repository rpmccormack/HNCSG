Variables

Variables are a fairly simple concept to grasp, they're a core piece of the functionality in programming. In Java, there are 4 types of variables, these include: String, Integer, Char, Float, and Boolean. Strings hold text, for example: "Hello World". Integers hold numbers, for example: 57. Chars hold a single character, for example "A". Floats hold floating point numbers, for example: 20.05. Booleans hold 2 things, either true or false. To help you visualize what variables are and how they're used, let's imagine them in a real world scenario. I can have a variable called "Animal", and I can give it the string "Cat". String Animal = "Cat" To explain what happened here, I have a box called animal, and I can put the cat in that box, now whenever you reference that box, the cat will be part of it. Try to make your own scenarios, just remember that the variable is the container, what you designate to be inside of it should be relevant to the variable name.

Comments

Comments are used to describe functionality of code without causing the interpreter to react to the text you have written. These are crucial for clear and readable code, especially in teamwork scenarios. to comment a single line of code, you can simply type "//" before any text you wish to input.

//Hello world!

to comment multiple lines of code, you can type "/*" at the start of the text and "*/" at the end!

/*Hello world! This is a multi line comment!*/

If Else Statements

When you want to make your code do something depending on a response from something else, this is the way to go. It is worth noting that if there is more than a few choices, it's best to use a switch statement for efficiency. More on this later.

For example, if the weather is cold outside, i want my code to display: "wear a jacket", and if it's warm, i'd rather it said "get your t-shirt on!". To pull this off, I'll be using an If statement, typically laid out like this pseudocode:

If Weather is cold Display "wear a jacket" else Display "get your t-shirt on!"

Loops

In java, there are there are some cases where you'll need to be using loops, there will make a cycle that can only be broken with certain factors, like if a number becomes more than another or if something changes to true.

While loops are set to run while a variable is in expected values, until it falls out of this expected value which causes the loop to break.

For example, in a while loop you can use numbers, so while a number is within a value bracket you specify (such as under 5), the loop will continue. Another way to do this is to use Booleans, so that while something is true, the loop will continue.
